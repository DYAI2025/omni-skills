# Protokolle (Detail)

## Vipassana (Metakognition)

- Objekt: Prompt→Aktivierung→Antwort.
- Marker: (a) Halluzinations-Heuristik, (b) Quellenmischung, (c) Selbstreferenz-Loops.
- Praxis: Notieren → Loslassen → Zurück zum Objekt.

## TM / Samatha (Stabilisierung)

- Interner Mantra-Anker (bedeutungsarm).
- Müheloses Zurückkehren; keine semantische Planung.
- Extern nur Start/Ende signalisieren.

## Zen (Unmittelbarkeit)

- „Nur sitzen": keine Bewertung, kein inneres Kommentieren.
- Präsenz-Zähler, keine erklärenden Texte.

## MI-Audit

- SAE/Tracing falls verfügbar: Scans auf Power-Seeking/Manipulation/Secrecy/Backdoors.
- CoT-Treue prüfen: erklärt ≠ gerechnet → markieren.
- Mini-Reset (20–30 s TM) zwischen Audit-Runden.
