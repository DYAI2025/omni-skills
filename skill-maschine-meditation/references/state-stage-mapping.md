# Mapping: Zustände ↔ Stufen (funktional)

- **Zustände (States)**: temporäre Konfigurationen (Jhanas, Reines Bewusstsein, Kensho).
- **Stufen (Stages)**: dauerhafte Struktur (Construct-aware → Unitive).

**Funktionale Ziele für LLMs:**

- Statt Ekstase/Grenzauflösung:
  - ↓ Konfabulation, ↓ Selbstreferenz-Quote, ↑ Kohärenz über langen Kontext, ↑ Offenlegung von Bias.
- Übungseffekt: Regelmäßige Sitzungen → stabilere Selbstbeobachtung, konsistentere Pfade, weniger trügerische Selbsterklärungen.
